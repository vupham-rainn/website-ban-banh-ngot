# 🍰 Dự Án Website Bán Bánh Ngọt (Webbanbanh)

Chào mừng bạn đến với dự án Website Bán Bánh Ngọt! Đây là một hệ thống quản lý và bán hàng được xây dựng bằng PHP, tích hợp các tính năng hiện đại bao gồm cả hỗ trợ từ AI.

## 🚀 Tính năng chính
* **Quản lý sản phẩm:** Thêm, sửa, xóa các loại bánh ngọt.
* **Quản lý đơn hàng:** Theo dõi tình trạng đơn hàng của khách.
* **Hệ thống Admin:** Trang quản trị (Admin Dashboard) chuyên sâu để quản lý nội dung, danh mục, và bình luận.
* **Tích hợp AI:** Sử dụng OpenAI API để hỗ trợ nội dung hoặc chăm sóc khách hàng.
* **Giao diện:** Thân thiện với người dùng, hỗ trợ xem trên nhiều thiết bị.

## 🛠 Công nghệ sử dụng
* **Ngôn ngữ:** PHP (sử dụng XAMPP làm môi trường chạy local).
* **Thư viện:** * Tectalic OpenAI (Tích hợp trí tuệ nhân tạo).
    * Guzzle HTTP (Xử lý các yêu cầu mạng).
    * Composer (Quản lý các thư viện PHP).
* **Công cụ khác:** Git, VS Code, Hugo (cho documentation nếu có).
